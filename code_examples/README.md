# AI & Deep Learning Concepts - Code Examples & Scripts

This repository contains practical implementations and tutorials for all concepts covered in the IROST presentations:
1. **application.html** - Deep Learning Revolution (DNNs, CNNs, RNNs, GANs, Transformers, etc.)
2. **presentation.html** - Generative AI Overview (LLMs, Prompt Engineering, AI Tools, Automation)

## Directory Structure

```
code_examples/
├── 1_fundamentals/                    # AI & ML Basics
├── 2_deep_learning/                   # Neural Networks Fundamentals
├── 3_cnns/                           # Convolutional Neural Networks
├── 4_rnns_lstms/                     # Recurrent Networks & LSTMs
├── 5_generative_models/              # GANs & VAEs
├── 6_transformers/                   # Transformer Architecture
├── 7_diffusion_models/               # Diffusion Models
├── 8_large_language_models/          # LLMs & GPT
├── 9_prompt_engineering/             # Prompt Engineering Techniques
├── 10_multimodal_models/             # Vision & Language Models
├── 11_reinforcement_learning/        # Deep RL
├── 12_ai_tools_integration/          # API Integration Examples
├── 13_automation_workflows/          # Automation & No-Code Tools
└── 14_deployment_optimization/       # Model Deployment & Optimization
```

## Quick Start

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Explore Each Topic**
   - Navigate to each directory
   - Read the `README.md` for concept explanation
   - Run the example scripts
   - Modify and experiment!
